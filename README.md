# Simple CRUD App

This is a basic CRUD (Create, Read, Update, Delete) application built with **HTML**, **CSS**, and **JavaScript**. It allows users to add, edit, delete, and view items in a simple UI — all stored in the browser using **localStorage**.

## 💻 Features

- 📝 Add new items
- 📋 View the list of items
- ✏️ Edit existing items
- 🗑️ Delete items
- 💾 Data saved locally in the browser

## 🧰 Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
- localStorage API

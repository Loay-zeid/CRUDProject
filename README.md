# 🎨 CRUD Application (Frontend)

## 📌 Overview

The **CRUD Application** is a simple and interactive frontend project built using **HTML, CSS, and JavaScript**. It allows users to manage data through core operations including creating, reading, updating, and deleting records.

Designed as a foundational project, it focuses on mastering DOM manipulation, event handling, and local data persistence without relying on external frameworks.

---

## 🚀 Live Demo

🔗 https://loay-zeid.github.io/CRUDProject/

---

## 🎯 Key Features

- ➕ Create new records through a form interface  
- 📋 Display records dynamically in a structured table  
- ✏️ Update existing entries   
- ❌ Delete records instantly  
- 💾 Persistent data storage using Local Storage  
- ⚡ Fast and responsive interaction  

---

## 🧰 Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla)
